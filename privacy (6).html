1-3. AI 분석 처리 (OpenAI GPT-4o mini)

악플AI는 댓글 분석에 OpenAI의 GPT-4o mini API를 사용합니다. 댓글 본문은 분석 목적으로 OpenAI API에 전송되며, OpenAI의 데이터 처리 정책이 적용됩니다. 전송 시 이용자의 개인 식별 정보(이름, 이메일 등)는 포함되지 않습니다. OpenAI 개인정보처리방침: openai.com/privacy

2. 개인정보 처리 방법

  • 수집된 댓글 데이터는 악플 탐지 및 자동 대응 목적으로만 사용됩니다.
  • Google OAuth 인증을 통해 접근하며, 비밀번호는 수집하지 않습니다.
  • 요청 권한은 댓글 읽기(youtube.readonly) 및 댓글 쓰기(youtube.force-ssl) 최소 권한만 요청합니다.
  • 모든 저장 데이터는 AES-256 암호화로 보호됩니다.
  • 자동 대댓글 기능은 이용자가 직접 활성화한 경우에만 작동합니다.

3. 개인정보 제3자 제공

악플AI는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의거하거나 수사기관의 적법한 요청이 있는 경우
  • Elite 플랜의 법무법인 연계 서비스 이용 시 (이용자의 별도 동의 필요)

6. 베타 서비스 관련

현재 서비스는 베타 운영 중이며, 베타 기간 동안 수집된 피드백 및 사용 데이터는 서비스 개선 목적으로 활용될 수 있습니다. 베타 종료 시 별도 공지를 통해 데이터 처리 방침을 안내합니다.

7. 증거저장 기능 및 고소장 생성

7-1. 증거저장 기능

항목내용
수집 항목댓글 본문, 작성자 채널 ID, 댓글 URL, 작성 일시, 페이지 스크린샷

증거저장 기능은 댓글 작성자(제3자)의 공개된 댓글 데이터를 수집합니다. 수집된 자료는 이용자(채널 소유자)의 법적 대응 지원 목적으로만 사용되며, 제3자에게 제공되지 않습니다. 단, 실제 법적 절차에서 해당 자료의 증거 능력은 관할 법원의 판단에 따릅니다.

7-2. 고소장 초안 생성 기능 (Elite 플랜)

⚠️ 중요: 고소장 초안 생성은 AI가 수집된 증거를 바탕으로 문서 초안을 작성하는 기능입니다. 생성된 문서는 법적 효력을 보장하지 않으며, 실제 법적 효력을 위해서는 반드시 변호사의 검토가 필요합니다. 악플AI는 법률 서비스를 제공하지 않습니다.

  • 고소장 초안은 GPT-4o mini AI가 증거 데이터를 기반으로 자동 생성합니다.
  • 생성된 초안은 이용자 기기 및 서비스 내에 저장되며, 이용자 요청 시 즉시 삭제됩니다.
  • 초안 생성 시 수집된 증거 데이터가 OpenAI API에 전송될 수 있습니다 (개인 식별 정보 제외).
  • 악플AI 제휴 법무법인 연계 서비스는 이용자의 별도 동의 후 진행됩니다.
  • 법무법인 연계 시 필요한 증거 자료 전달은 이용자의 명시적 동의 후에만 이루어집니다.

8. 쿠키 및 분석 도구

  • Google Analytics 4를 통한 방문자 분석 (익명화 처리)
  • 브라우저 localStorage를 통한 사용자 설정 저장 (개인 식별 불가)
  • 브라우저 설정에서 쿠키를 거부할 수 있으나, 일부 서비스 이용이 제한될 수 있습니다.

9. 보안 조치

  • AES-256 암호화로 저장 데이터 보호
  • HTTPS(TLS 1.3) 통신 암호화
  • OAuth 토큰은 최소 권한 원칙에 따라 관리
  • 정기적인 보안 점검 실시
