1 수집하는 개인정보 항목

Creator Shield(이하 "서비스")는 YouTube 크리에이터의 채널을 악성댓글로부터 보호하기 위해 아래와 같은 정보를 수집합니다.

수집 항목 수집 방법 필수 여부
이메일 주소, 이름 회원가입 시 직접 입력 또는 Google 로그인 필수
YouTube 채널명, 채널 ID, 구독자 수, 썸네일 YouTube OAuth 인증 후 YouTube API 자동 수집 필수
YouTube 댓글 내용, 작성자 ID, 작성 시간, 영상 URL YouTube API 자동 수집 필수
서비스 이용 기록, 접속 로그, IP 주소 서비스 이용 시 자동 수집 자동
결제 정보 (카드번호 제외) 유료 플랜 가입 시 토스페이먼츠를 통해 수집 선택
⚠️ 수집하지 않는 정보 비밀번호, 카드 전체 번호, 주민등록번호 등 민감한 개인정보는 수집하지 않습니다. YouTube 계정 비밀번호는 절대 요구하지 않습니다.

2 개인정보 수집 및 이용 목적

수집한 정보는 오직 아래 목적으로만 사용되며, 목적이 변경될 경우 별도 동의를 받습니다.

  • AI 기반 악성댓글 자동 감지 및 분류
  • 채널 위험도 점수 및 클린 점수 산출
  • 사용자 설정에 따른 댓글 자동 숨김 처리
  • 법적 증거자료 보존 및 고소장 초안 생성 (Elite 플랜)
  • 월간 채널 건강 리포트 제공
  • 서비스 개선 및 오류 분석 (익명화 처리)
  • 유료 플랜 결제 처리 및 청구
  • 고객 문의 응대 및 공지 발송

3 Google API 데이터 사용 정책

Creator Shield의 Google API 사용 및 데이터 처리는 Google API Services User Data Policy를 완전히 준수합니다.

🔒 Limited Use 요건 준수 Google API를 통해 수집한 데이터는 서비스 제공에 필요한 용도로만 사용됩니다. 광고 목적, 제3자 판매, 크레딧 평가, 채용, 보험 등에 절대 사용하지 않습니다.

사용하는 YouTube API 권한(Scope):

  • youtube.readonly — 채널 및 댓글 정보 읽기 전용 접근
  • youtube.force-ssl — 크리에이터 요청 시 악성댓글 숨김 처리
🔗 Google API Services User Data Policy: developers.google.com

Google 계정 권한 관리 및 연결 해제는 언제든지 가능합니다:

🔗 myaccount.google.com/permissions

5 개인정보의 제3자 제공

Creator Shield는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 단, 아래의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 따라 수사기관 등이 요구하는 경우
  • Elite 플랜 법무법인 연결 서비스 이용 시 (별도 동의 후 해당 법무법인에만 제공)
🚫 절대 하지 않는 것 Google API를 통해 수집한 사용자 데이터는 광고에 활용하거나 제3자에게 판매하지 않습니다.

6 개인정보 보호 조치

  • OAuth 토큰 AES-256 암호화 저장
  • HTTPS(SSL/TLS) 전송 구간 암호화
  • 접근 권한 최소화 (필요한 직원에게만 접근 허용)
  • 정기적 보안 취약점 점검
  • 개인정보 취급자 교육 실시

7 이용자의 권리

이용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청 — 서비스 내 "설정 → 내 정보" 메뉴
  • 개인정보 수정 요청 — 서비스 내 직접 수정 또는 이메일 요청
  • 개인정보 삭제 요청 — 회원 탈퇴 또는 이메일 요청 (30일 내 처리)
  • YouTube 연결 해제 — 서비스 내 "설정 → 채널 연결 해제"
  • 처리 정지 요청 — privacy@creatorshield.kr로 이메일 요청

8 쿠키 사용

서비스는 로그인 상태 유지 및 서비스 개선을 위해 쿠키를 사용합니다. 브라우저 설정에서 쿠키를 거부할 수 있으나, 일부 서비스 이용이 제한될 수 있습니다.

중요한 변경의 경우 이메일을 통해 별도 안내합니다.
